Introduction

Termology

  • Script; Command; Comment; Variable; Fun; Argument

  • Do I have to set/enter all arguments?

Synatax

  • A set of rules

Packages

  • A collection of R funs
  • Base: function name{base}
  • Base cheat sheet
    • http://www.rstudio.com/wp-content/uploads/2016/10/r-cheat-sheet-3.pdf
  • Install packages
    • Install.package()
    • Library()
    • [packagename]::[fun] eg. devtools::check()
  • Where can I find R packages?
    • CRAN
    • Bioconduct ???
    • Github
  • Most used packages
    • https://towardsdatascience.com/top-100-most-used-r-functions-on-github-9caf2b81b314

Working directory

R project

  • A collection of files (data; results; graphs)
  • To keep your analysis organized
    • Data; papers; rcode; writing; Rproj; Readme
  • Relative path: use to do reproductive work and share
  • What files do you have in this folder?
    • .Rhistory
    • .Rproj
    • Auto-saved

Help in R

  • ?funname
  • ?”special characters”
  • help(package = “package name”)
  • example(function name)

Data Type

  • Most common atomic data type
    • Numeric: 1, 2.3, 14.78
    • Integers
    • Complex
    • Logical
    • Characters
  • Find out the data type
    • class(2)
    • class(4L)
    • class(1+2i)
    • class(TRUE)
  • Change data type later too
    • as.numeric()
    • Remember some cant be changed or in an error way!!!

Data structure

  • Data Frames
    • Create a data frame
    • data.frame()
    • Load mtcars data frame
      • data() # to see the different datasets within R
      • ?mtcars
    • Useful commands
      • str() # find out the stucture of a data frame
      • head()
      • head(mtcars,10) # list the first 6 rows of data (or you could specific how many rows you want to preview)
      • tail() # list the last 6 rows of data (or you could specific how many rows you want to preview)
    • Subset
      • [row, column]
      • [“var name”]
      • df$var
        ® mtcars[3,2]
® mtcars[3, ]
® mtcars[3:7,2]
      • Use subset() to select data (ex: mpg >15)
        ® subset(mtcars, mpg > 15)
